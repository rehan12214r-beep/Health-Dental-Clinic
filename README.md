# Health-Dental-Clinic

A modern healthcare and dental clinic website featuring appointment booking, medical services, and patient information.

## Features

- Responsive design for all devices
- Modern UI with clean aesthetics
- Medical and dental service listings
- Appointment booking system
- Patient testimonials
- Doctor profiles
- Contact and location information

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
